# 按钮权限配置清单

## 权限代码规范

所有权限代码统一使用下划线分隔格式：`{模块}_{操作}`

### 通用操作权限代码

| 操作 | 权限代码 | 说明 |
|------|---------|------|
| 新增 | `{module}_add` | 创建新记录 |
| 编辑 | `{module}_edit` | 修改现有记录 |
| 删除 | `{module}_delete` | 删除记录 |
| 查看 | `{module}_view` | 查看详情 |
| 查询 | `{module}_query` | 查询列表 |
| 导出 | `{module}_export` | 导出数据 |
| 导入 | `{module}_import` | 导入数据 |
| 审核 | `{module}_audit` | 审核记录 |
| 配置 | `{module}_config` | 配置功能 |

## 权限实现模式

### 标准实现方式

```tsx
import { usePageButtons } from '@/hooks/usePageButtons';
import { PermissionButton, checkPermission } from '@/components/PermissionButton';

const MyPage: React.FC = () => {
  const { buttons } = usePageButtons();

  return (
    <PageContainer
      extra={buttons
        .filter(btn => btn.action === 1)
        .map(btn => <Button key={btn.code}>{btn.name}</Button>)
      }
    >
      <ProTable
        toolBarRender={() => [
          <PermissionButton
            hasPermission={checkPermission(buttons, 'my_add')}
            noPermissionTip="您没有新增权限"
          >
            <Button type="primary" icon={<PlusOutlined />}>新增</Button>
          </PermissionButton>,
          <PermissionButton
            hasPermission={checkPermission(buttons, 'my_delete')}
            showWhenNoPermission
            noPermissionTip="您没有删除权限"
          >
            <Button danger icon={<DeleteOutlined />}>删除</Button>
          </PermissionButton>,
        ].filter(Boolean)}
      />
    </PageContainer>
  );
};
```

### 简化实现方式

```tsx
import { usePageButtons } from '@/hooks/usePageButtons';

const MyPage: React.FC = () => {
  const { buttons } = usePageButtons();

  return (
    <ProTable
      toolBarRender={() => [
        buttons.some(btn => btn.code === 'my_add') && (
          <Button type="primary">新增</Button>
        ),
        buttons.some(btn => btn.code === 'my_delete') && (
          <Button danger>删除</Button>
        ),
      ].filter(Boolean)}
    />
  );
};
```
