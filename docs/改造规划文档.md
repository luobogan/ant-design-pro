## 1. 改造背景与目标

### 1.1 改造背景

当前 `ant-design-pro` 项目基于 UmiJS 4 + React 19 + TypeScript 构建，采用现代化的 Hooks 架构。参考 Sword 项目的权限控制与通用组件设计规范，对项目进行优化改造。

### 1.2 改造目标

**总体目标**：借鉴 Sword 项目的权限控制与通用组件设计规范，优化 `ant-design-pro` 项目，提升代码复用率和开发效率。

**具体目标**：

1. **权限控制完善**：建立完整的 RBAC 权限模型，实现页面权限、按钮权限的全面控制
2. **通用组件封装**：开发 Panel、ToolBar、SearchBox、Grid 等通用组件，提升组件复用率
3. **文件组织规范化**：建立统一的文件命名规范和代码组织方式
4. **开发效率提升**：新增业务模块的开发时间缩短 30% 以上

### 1.3 改造原则

1. **渐进式改造**：分阶段、分模块逐步推进，避免一次性大规模改动
2. **向后兼容**：保证改造过程中现有功能的正常使用
3. **现代化架构**：继续使用 UmiJS 4 + React 19 + Hooks，不引入 Dva
4. **可维护性**：代码结构清晰，易于理解和维护

## 4. 权限控制模块改造方案

### 4.1 权限模型设计

#### 4.1.1 RBAC 模型

采用基于角色的权限控制（RBAC）：

```
用户 (User)
  ↓
角色 (Role)
  ↓
权限 (Permission)
  ├── 页面权限
  └── 按钮权限
```

### 4.2 权限存储方案

#### 4.2.1 存储内容

```typescript
// 权限存储键名
const TOKEN_KEY = 'sword-token';
const ACCESS_TOKEN_KEY = 'sword-access-token';
const AUTHORITY_KEY = 'sword-authority';
const USER_INFO_KEY = 'sword-user-info';
const BUTTONS_KEY = 'sword-buttons';
```

#### 4.2.2 权限工具函数

```typescript
// src/utils/authority.ts
export function getAuthority(): string[]
export function getButtons(): any[]
export function getButton(code: string): any[]
export function hasAuthority(authorityCode: string): boolean
export function hasButton(code: string): boolean
```

### 4.3 权限验证流程

#### 4.3.1 路由权限验证

```
访问路由
  ↓
检查路由配置
  ↓
验证用户权限（access.ts）
  ↓
有权限 → 渲染页面
无权限 → 跳转 403 页面
```

#### 4.3.2 按钮权限验证

```
页面加载
  ↓
获取页面 code
  ↓
调用 getButton(code)
  ↓
过滤无权限按钮
  ↓
渲染按钮
```

## 5. 通用组件整合策略

### 5.1 BusinessComponents 组件库

#### 5.1.1 Panel 组件

**功能：** 页面面板容器

**使用示例：**
```tsx
import { Panel } from '@/components/BusinessComponents';

<Panel title="用户管理" back="/system/user">
  <div>页面内容</div>
</Panel>
```

#### 5.1.2 ToolBar 组件

**功能：** 工具栏按钮渲染

**使用示例：**
```tsx
import { ToolBar } from '@/components/BusinessComponents';

<ToolBar
  buttons={buttons}
  onClick={handleButtonClick}
/>
```

#### 5.1.3 SearchBox 组件

**功能：** 搜索表单封装

**使用示例：**
```tsx
import { SearchBox } from '@/components/BusinessComponents';

<SearchBox onSubmit={handleSearch} onReset={handleReset}>
  <Input name="keyword" placeholder="请输入关键词" />
</SearchBox>
```

#### 5.1.4 Grid 组件

**功能：** 通用表格组件

**使用示例：**
```tsx
import { Grid } from '@/components/BusinessComponents';

<Grid
  code="system.user"
  columns={columns}
  data={userData}
  loading={loading}
  onSearch={handleSearch}
/>
```

### 5.2 组件使用规范

1. **统一导入**：从 `@/components/BusinessComponents` 导入
2. **类型安全**：使用 TypeScript 类型定义
3. **权限集成**：组件内置权限控制支持
4. **文档完善**：参考 README.md 使用文档
