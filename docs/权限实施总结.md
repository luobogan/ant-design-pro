### 1. 核心基础设施

#### 1.1 usePageButtons Hook
**文件**: `src/hooks/usePageButtons.ts`

**功能特性**:
- ✅ 自动从路由路径提取菜单 code（如 `/system/user` → `'user'`）
- ✅ 支持手动指定菜单 code
- ✅ 返回按钮权限数组和加载状态
- ✅ 内置调试日志
- ✅ 权限动态加载，无需刷新页面

**使用示例**:
```tsx
const { buttons, loading } = usePageButtons();
// 自动从 /system/user 提取 'user'，获取用户管理页面的按钮权限
```

#### 1.2 PermissionButton 组件
**文件**: `src/components/PermissionButton/index.tsx`

**功能特性**:
- ✅ 统一的权限控制按钮组件
- ✅ 支持无权限时隐藏模式
- ✅ 支持无权限时禁用模式（带 Tooltip 提示）
- ✅ 完全继承 antd Button 的所有属性
- ✅ 提供权限检查辅助函数

**使用示例**:
```tsx
<PermissionButton 
  hasPermission={buttons.some(btn => btn.code === 'user_add')}
  showWhenNoPermission
  noPermissionTip="您没有新增权限"
>
  <Button type="primary">新增</Button>
</PermissionButton>
```

## 🔧 实施细节

### 标准实施模式

每个页面的权限控制实施遵循以下标准步骤：

#### 步骤 1: 导入 Hook
```tsx
import { usePageButtons } from '@/hooks/usePageButtons';
```

#### 步骤 2: 获取按钮权限
```tsx
const { buttons } = usePageButtons();
```

#### 步骤 3: 工具栏按钮权限控制
```tsx
toolBarRender={() => [
  buttons.some(btn => btn.code === 'module_add') && (
    <Button type="primary" icon={<PlusOutlined />}>
      新增
    </Button>
  ),
  buttons.some(btn => btn.code === 'module_delete') && (
    <Button danger icon={<DeleteOutlined />}>
      删除
    </Button>
  ),
].filter(Boolean)}
```

#### 步骤 4: 操作列按钮权限控制
```tsx
{
  title: '操作',
  render: (_: any, record) => (
    <Space>
      {buttons.some(btn => btn.code === 'module_edit') && (
        <Button type="link" icon={<EditOutlined />}>编辑</Button>
      )}
      {buttons.some(btn => btn.code === 'module_delete') && (
        <Button type="link" danger icon={<DeleteOutlined />}>删除</Button>
      )}
    </Space>
  ),
}
```

#### 步骤 5: PageContainer.extra 权限控制
```tsx
<PageContainer
  extra={buttons
    .filter(btn => btn.action === 1)
    .map(btn => <Button key={btn.code}>{btn.name}</Button>)
  }
>
  {/* 页面内容 */}
</PageContainer>
```

## 🚀 后续优化建议

### 1. 权限刷新机制
当前权限在应用启动时加载，建议添加权限刷新机制：

```tsx
// 在 usePageButtons 中添加
const refreshPermissions = async () => {
  await fetchButtons();
  setInitialState(s => ({ ...s, buttons: newButtons }));
};
```

### 2. 权限缓存策略
添加缓存过期时间，避免频繁请求：

```tsx
const CACHE_EXPIRY = 30 * 60 * 1000; // 30 分钟
```

### 3. 权限降级处理
API 失败时使用缓存数据：

```tsx
const cached = getButtons();
if (cached && cached.length > 0) {
  return cached;
}
```

### 4. 批量权限检查工具
创建批量权限检查工具函数：

```tsx
// 检查是否有任一权限
const hasAnyPermission = (buttons: any[], codes: string[]) => {
  return codes.some(code => buttons.some(btn => btn.code === code));
};
```

## 📌 注意事项

### 1. 权限代码命名
- ✅ 使用下划线分隔：`user_add`
- ❌ 不使用冒号分隔：`user:add`
- ❌ 不使用驼峰命名：`userAdd`

### 2. 权限判断位置
- ✅ 工具栏按钮：使用 `buttons.some()` 判断
- ✅ 操作列按钮：使用 `buttons.some()` 判断
- ✅ PageContainer.extra：使用 `buttons.filter()` 过滤
